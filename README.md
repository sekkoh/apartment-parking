# 駐車場代表示

賃貸一覧画面上で駐車場代を表示する Chrome 拡張機能。Suumo・エイブル対応。

<img width="1340" height="931" alt="スクリーンショット 2026-03-30 23 49 22" src="https://github.com/user-attachments/assets/257b1b0d-804a-4e9a-86d7-29532cfca13a" />


## インストール

### crxファイルより
releaseよりcrxファイルをDLする

### リポジトリより

1. このリポジトリをクローンまたは ZIP でダウンロード
2. Chrome で `chrome://extensions/` を開く
3. 「デベロッパーモード」を有効化
4. 「パッケージ化されていない拡張機能を読み込む」からこのフォルダを選択

## 対応サイト

- [Suumo](https://suumo.jp/) 賃貸一覧
- [エイブル](https://www.able.co.jp/) 賃貸一覧
